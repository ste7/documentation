## Welcome

Welcome to the Novacal API documentation. Novacal is a powerful meeting scheduling software that helps you manage calendars, book appointments, and integrate with Google Calendar, Outlook, Zoom, and more.

Our REST API allows you to programmatically manage event types, events, teams, and user information to build custom scheduling workflows and integrate Novacal into your applications.

## Authentication

All API endpoints require authentication using Bearer tokens. Make sure to include your API key in the Authorization header for all requests.

```bash
Authorization: Bearer YOUR_API_TOKEN
```
