---
title: "Get Teams"
openapi: "GET /v1/teams"
---

Retrieve all teams that the authenticated user is a member of.

<Note>
  Users can only be members of one team. If you already have a team, you cannot
  create another one.
</Note>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the request was successful. Always `true` for successful
  responses.
</ResponseField>

<ResponseField name="data" type="array">
  Array of team objects.
  
  <Expandable title="Team properties">
    <ResponseField name="id" type="integer" required>
      The unique identifier of the team.
    </ResponseField>

    <ResponseField name="name" type="string" required>
      The name of the team (max 55 characters).
    </ResponseField>

    <ResponseField name="username" type="string" required>
      The unique username for the team (max 255 characters, alphanumeric with hyphens and underscores).
    </ResponseField>

    <ResponseField name="about" type="string">
      Description or about text for the team (max 5000 characters).
    </ResponseField>

    <ResponseField name="logo" type="string">
      URL to the team's logo image. Returns `null` if no logo is set.
    </ResponseField>

  </Expandable>
</ResponseField>
