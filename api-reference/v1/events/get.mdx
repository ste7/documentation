---
title: "Get Events"
openapi: "GET /v1/events"
---

Retrieve all events that the authenticated user has access to. This endpoint returns a list of events owned by the user or accessible through team memberships.

## Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the request was successful. Always `true` for successful
  responses.
</ResponseField>

<ResponseField name="data" type="array">
  Array of event objects.
  
  <Expandable title="Event properties">
    <ResponseField name="id" type="integer" required>
      The unique identifier of the event.
    </ResponseField>

    <ResponseField name="start" type="string" required>
      The start date and time of the event in ISO 8601 format.
    </ResponseField>

    <ResponseField name="end" type="string" required>
      The end date and time of the event in ISO 8601 format.
    </ResponseField>

    <ResponseField name="location" type="string">
      The location of the event. Can be a physical address, video call link, or other location information.
    </ResponseField>

    <ResponseField name="status" type="string" required>
      The status of the event. Possible values: `scheduled`, `canceled`, `completed`.
    </ResponseField>

  </Expandable>
</ResponseField>
