## Welcome

Welcome to the Novacal API documentation. This REST API allows you to programmatically manage event types, events, teams, and user information. All endpoints require authentication using Bearer tokens and return responses in JSON format.

Use this API to integrate Novacal's scheduling capabilities into your applications, automate event management, and build custom scheduling workflows.
