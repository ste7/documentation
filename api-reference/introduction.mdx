## Event Types

Event types define the different kinds of meetings or events that can be scheduled. Each event type has properties such as:

- **Name and slug**: Human-readable name and URL-friendly identifier
- **Type**: One of `one_on_one`, `group`, `round_robin`, or `collective`
- **Duration**: Meeting length in minutes (1-1440)
- **Group size**: Maximum participants for group events (2-10)
- **Color**: Visual identifier in hex format
- **Visibility**: Whether shown on public profile

## Teams

Teams allow multiple users to collaborate on event types and events. Each team has:

- **Name and username**: Human-readable name and unique identifier
- **About**: Description of the team
- **Logo**: Team branding image
