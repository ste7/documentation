---
title: "Get Team"
openapi: "GET /v1/teams/{id}"
---

Retrieve a single team by its ID. You can only access teams that you are a member of.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the team.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the request was successful. Always `true` for successful
  responses.
</ResponseField>

<ResponseField name="data" type="object">
  The team object.
  
  <Expandable title="Team properties">
    <ResponseField name="id" type="integer" required>
      The unique identifier of the team.
    </ResponseField>

    <ResponseField name="name" type="string" required>
      The name of the team (max 55 characters).
    </ResponseField>

    <ResponseField name="username" type="string" required>
      The unique username for the team (max 255 characters).
    </ResponseField>

    <ResponseField name="about" type="string">
      Description or about text for the team (max 5000 characters).
    </ResponseField>

    <ResponseField name="logo" type="string">
      URL to the team's logo image. Returns `null` if no logo is set.
    </ResponseField>

  </Expandable>
</ResponseField>
